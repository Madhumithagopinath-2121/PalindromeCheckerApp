# PalindromeCheckerApp
//I USED JAVA PROGRAMMING LANGUAGE TO WRITE THIS CODE -UC7
//The PalindromeChecker App (UC7) is a console-based Java application that checks whether a given string is a palindrome using a Deque (Double Ended Queue).
//This use case demonstrates how front and rear access can be used efficiently to validate palindrome logic without using separate Stack and Queue structures.
//1. Read input string from user
//2. Convert to lowercase
//3. Remove special characters
//4. Insert characters into Deque
//5. Remove first and last characters
//6. Compare them
//7. Repeat until:
    - Deque is empty → Palindrome
    - Mismatch occurs → Not Palindrome
